# DioneJuliaPelicano_hello_world
This project is created using python for IS 312 Activity 3. When accessed through a browser and the program runs, it displays "Hello World! Dione Julia Pelicano" message.

FILE STRUCTURE

Flask - a simple python tool used to build for building web applications.

HOW TO RUN APPLICATION LOCALLY
1. Install Python
2. Open terminal in the project folder
3. Make sure flask is installed
4. Run the application app.py file
5. Open a web browser and go to http://127.0.0.1:5000
7. The message "Hello World! Dione Julia Pelicano"  will appear
